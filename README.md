# Smart Nearby Places

A beginner-friendly web application that recommends nearby places based on the user's mood.

## Features
- Simple React-based UI
- Mood selection (work, date, quick bite, budget)
- Clean and beginner-readable code

## Tech Stack
- React
- JavaScript
- HTML & CSS

## How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000
