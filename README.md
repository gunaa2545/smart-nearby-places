# Smart Nearby Places
## Live Demo

Frontend: https://smart-nearby-places.netlify.app  
Backend API: https://smart-nearby-places-backend.onrender.com

## Architecture

This is a full-stack web application.

- Frontend: React (UI, mood selection, rendering)
- Backend: Node.js + Express (API abstraction)
- Data: Mocked Google Places API (used to avoid exposing API keys)

The backend is designed to easily switch to the real Google Places API when billing is enabled.
A beginner-friendly web application that recommends nearby places based on the user's mood.

## Features
- Simple React-based UI
- Mood selection (work, date, quick bite, budget)
- Clean and beginner-readable code

## Tech Stack
- React
- JavaScript
- HTML & CSS

## How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000
